# Simple Alarm Clock in C

A console-based alarm clock program built using C.

## Features
- Set alarm time (HH:MM)
- Display current time
- Trigger alarm message
- Exit program after alarm

## How to Run
1. Compile the program:

    gcc alarm_clock.c -o alarm_clock

2. Run:

    ./alarm_clock
